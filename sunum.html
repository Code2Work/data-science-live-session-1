SQL ile Veriyi Tanımak

pgAdmin'de canlı sorgular

Bölüm 1 — SQL

Temel Sorgular

-- Kaç ürünümüz var?
SELECT COUNT(*) FROM products;

-- Kategorilere göre ürün sayısı
SELECT c.category_name, COUNT(p.product_id) AS urun_sayisi
FROM products p
JOIN categories c ON p.category_id = c.category_id
GROUP BY c.category_name
ORDER BY urun_sayisi DESC;
Bölüm 1 — SQL

Aylık Satış Trendi

SELECT
  DATE_TRUNC('month', o.order_date) AS ay,
  ROUND(CAST(SUM(od.unit_price * od.quantity) AS numeric), 2) AS toplam_satis
FROM orders o
JOIN order_details od ON o.order_id = od.order_id
GROUP BY ay
ORDER BY ay;

CAST(... AS numeric) — tip dönüşümü yapar.
ROUND() fonksiyonu numeric tip bekler, bu yüzden önce dönüştürüyoruz.

Her Sorgudan Sonra Durun ve Sorun:

  • "Bu bize ne anlatıyor?"
  • "Beverages en çok ürüne sahip kategori — neden önemli?"
  • "Aylık satışlarda bir düşüş var mı? Nerede?"
  • "En çok sipariş veren müşteri kimdir?"

Veri analisti sadece sorgu yazmaz — soru sorar.

05

NumPy — Sayısal
Hesaplamanın Temeli

Pandas'ın altında çalışan motor

Bölüm 5 — NumPy

NumPy Array vs Python Listesi

import numpy as np

fiyatlar = np.array([10, 20, 30, 40, 50])

# Tek seferde tüm elemanlara işlem
print("KDV ekle:", fiyatlar * 1.18)
print("Toplam:", fiyatlar.sum())
print("Ortalama:", fiyatlar.mean())

Python listesiyle bunu yapmak için döngü yazmanız gerekir.
NumPy ile tek satırda halledersiniz — ve 10-100x daha hızlı!

Bölüm 5 — NumPy

Gerçek Veri Üzerinde İstatistikler

satislar = df_orders['toplam'].values  # NumPy array!

print("Ortalama:", np.mean(satislar))
print("Medyan:",   np.median(satislar))
print("Std sapma:", np.std(satislar))
print("Min:",      np.min(satislar))
print("Max:",      np.max(satislar))

Medyan < Ortalama ise → büyük siparişler ortalamayı yukarı çekiyor.
Çoğu sipariş aslında ortalamanın altında. Buna çarpık dağılım denir.

Bölüm 5 — NumPy

Yüzdelik Dilimler (Percentile)

q25, q50, q75 = np.percentile(satislar, [25, 50, 75])

print(f"Alt çeyrek (Q1):  ${q25:,.2f}")
print(f"Medyan (Q2):      ${q50:,.2f}")
print(f"Üst çeyrek (Q3):  ${q75:,.2f}")

%75'i Q3'ün altında

Tipik sipariş aralığını gösterir

Q3 üstündekiler → VIP

Bu müşterilere özel ilgi göster!
